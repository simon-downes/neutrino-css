RTL

Support for right-to-left text in Neutrino.

To enable RTL in Neutrino you need to set dir="rtl"on the <html> element (example).

<!doctype html>
<html dir="rtl" lang="ar">
  ...
</html>

ℹ️ The RTL feature is still experimental and will probably evolve.
